# Работа с накладными

:::note

В настоящий момент модель продаж [eDBS](https://seller.wildberries.ru/instructions/ru/ru/material/A-107) недоступна.
Заказы с данным типом автоматически синхронизироваться не будут.

:::

Сервис интеграции автоматически проверяет наличие новых сборочных заданий на Wildberries каждые 15 минут. Информация
о новых заданиях сохраняется в системе и ожидает дальнейшего перехода статуса задания.

:::warning

Для корректности работы, рекомендуется не передавать задания на сборку первые 30 минут после его появления на
Wildberries.
Это связано с техническим особенностями API интеграции с DBS.

:::note

Для модели продаж eDBS данного ограничения нет, поскольку сервис автоматически их исключает из синхронизации

:::

После перехода статуса задания на сборку, интеграция создает накладную в СДЭК.

При этом происходит консолидация заказов: если несколько сборочных заданий доставляются одному покупателю по одному
адресу, то они будут автоматически объединены в одну накладную.

Созданная накладная отобразиться в [Личном кабинете СДЭК](https://lk.cdek.ru) и будет доступна для редактирования.

:::info

В созданной накладной могут использоваться подменные артикулы, отличающиеся от артикула настоящего товара. Это не
является ошибкой и необходимо для корректной работы интеграции.

:::

:::danger

При редактировании распределения товара по упаковкам (создания новой упаковки с товарами или объединения двух
упаковок в одну) необходимо сохранять артикулы и количество, в противном случае автоматизация данного заказа
может перестать работать корректно

:::

После передачи накладной в СДЭК, статус связанных с ней сборочных заданий автоматически изменится на `В доставке`.

Поскольку Wildberries использует подменные номера телефонов, автоматическое уведомление покупателей от СДЭК
работать не будет.

:::tip

На данном этапе рекомендуется самостоятельно связаться с покупателем и сообщить трек-номер накладной, чтобы он мог
отслеживать доставку своего заказа.

:::

За час до вручения заказа, курьер СДЭК свяжется с покупателем по телефону, а при вручении запросит код подтверждения,
отображаемый в приложении Wildberries.

В зависимости от вручения или не вручения заказа, система передаст информацию об этом в Wildberries с кодом
подтверждения.

:::note

Если заказ будет отменен со стороны Wildberries, то накладной в СДЭК будет проставлен статус `Не вручен` и она будет
обработана в соответствии с правилами возврата по договору.

:::
