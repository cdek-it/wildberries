# Введение

# Предварительная настройка ЛК Wildberries
Перед настройкой интеграции CDEK с Wildberries по DBS выполните в ЛК Wildberries следующие шаги: 
- Создайте виртуальный склад DBS: «Поставки и заказы» → «Мои склады» → «Создать склад» → «Витрина (DBS)» по [инструкции WB](https://seller.wildberries.ru/instructions/ru/ru/material/how-to-create-warehouse-in-dbs).
- Настройте зону доставки по [инструкции](https://seller.wildberries.ru/instructions/ru/ru/material/how-to-create-warehouse-in-dbs?goBackOption=prevRoute&categoryId=e91121bf-4de0-425e-8d80-d416811cb571#%D1%81%D0%BE%D0%B7%D0%B4%D0%B0%D1%82%D1%8C-%D0%B7%D0%BE%D0%BD%D1%83-%D0%B4%D0%BE%D1%81%D1%82%D0%B0%D0%B2%D0%BA%D0%B8)
- Если необходимо настроить оплату за доставку с покупателей, это можно сделать по [инструкции](https://seller.wildberries.ru/instructions/ru/ru/material/paid-items-delivery-in-dbs-and-edbs?recommended=true#%D0%BA%D0%B0%D0%BA-%D1%83%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%82%D1%8C-%D0%BF%D0%BB%D0%B0%D1%82%D0%BD%D1%83%D1%8E-%D0%B4%D0%BE%D1%81%D1%82%D0%B0%D0%B2%D0%BA%D1%83)

После этого переходите к настройке интеграции. **Пожалуйста, не заводите товар на остатки пока интеграция не будет подключена — в противном случае автоматически накладная на заказ не создастся.**

Инструкция по настройке интеграции с магазина на Wildberries с сервисом доставки от СДЭК

:::info

Данная интеграция предназначена для автоматического создания накладных СДЭК для заказов, оформленных по модели DBS.
Подробнее про модель продаж DBS можно прочитать в [инструкции](https://seller.wildberries.ru/instructions/ru/ru/material/A-107).

:::

Основные особенности интеграции:
- Автоматическое создание накладных СДЭК для заказов на Wildberries без необходимости ручного переноса данных
- Синхронизация статуса доставки заказов между Wildberries и СДЭК
- Прогнозирование размера упаковки на основе товаров заказа и подбор оптимального тарифа для отправки

<details>
    <summary>Принцип работы интеграции</summary>

    После настройки, система каждые 15 минут проверяет наличие новых заказов на Wildberries.

    При обнаружении нового заказа, автоматически создается накладная в СДЭК.

    Накладная появляется в [личном кабинете](https://lk.cdek.ru), откуда ее можно распечатать и наклеить на посылку для
    передачи на доставку.
</details>

# Установка и настройка

Для работы интеграции, не требуется установка дополнительного ПО. Вся настройка происходит через личный кабинет СДЭК.

Для этого необходимо перейти на страницу [настроек интеграции](https://lk.cdek.ru/integration) и выбрать режим **DBS**.

<Image img={require('./img/start.png')} />

Далее необходимо сгенерировать токен Wildberries и ввести его в настройках интеграции.
Инструкцию по генерации токена можно прочитать на [отдельной странице](./wb-key.mdx).
